## N2N Client With GUI

![README中文版](./README_CN.md)

This is an n2n client with GUI, completely programmed and developed by **Claude Code**.

Written in **Python**, requires **Python 3.7+**

Solves the inconvenience of using n2n without a GUI.

## How to Use

- Download the `zip` and extract it to a separate folder
- Double-click `安装环境.bat` to install the environment
- Double-click `点我运行.bat` to launch the program
